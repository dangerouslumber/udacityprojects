Stage 0: Getting started with HTML

The World Wide Web

The internet is a collection of HyperText Markup Language (HTML) documents. HTML is the language that breowers your to decipher web pages. These different pages are connected with hyperlinks which create the web in the World Wide Web.

The Web is made up of 4 main components:

  • Browsers - programs that run on your computer to display HTML files
  • Internet - a vast network of computers
  • Servers - computers that host files that make up the web
  • HTTP - The main protocol of the web that browsers use to communicate with servers

HMTL Basics

HTML determines what you see on a website, what it looks like, and includes Links to other pages and references to other documents.

Markup, Tags, all good things

Markup is made up of tags and follow the pattern of opening tag contents closing tag. With these tags you can emphasize text or embolden it.

It is important to remember to end tags, because you could want this but end up with that instead. When writing code, remember that computers rely on specific instructions. Even small errors - like forgetting an end tag - can result in problems.

HTML Documents

When creating a HTML document, you need the following:
  1. Doctype
  2. HTML tags at the beginning and end of the document
  3. Head tag - contains the title, CSS, metadata, and other pieces
  4. Body tag - the contents of the document

Stage 1: Make a Web Page

Why CSS and HTML and how to think about them

Front-end development is a great place to begin learning. Students are able to receive immediate and visual feedback of their work.

When developers are given a mock-up of a design, they have to translate a visual into an actual website. One way to think about the makeup of a site is to compare it to a house. HTML is the walls of the house, CSS is the interior design and style, and Java Script is the interactive features of the house.

HTML - CSS - DOM

HTML is a language with snytax and rules and made up of tags. These tags are turned into elements in a tree by DOM - which is a standard convention for representing and interacting with elements in HTML. CSS determines the styling of the HTML - font, color, size, etc.

Boxes, Grids, and Rules

Website designs are based on boxes, and when creating a site from a visual, it is critical to unpack a design with boxes. Div tags help build boxes on sites and structure design.
